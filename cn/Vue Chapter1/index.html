  • DOM

Template Syntax

vue的模版语法是基于HTML的，但是在底层机制中，会将模版编译成高度优化的JS代码，因此可以通过：

  • HTML style 默认
  • JSX, 编写渲染函数而不使用模版，但是这样不会享受 VUE 自带的优化。

声明式渲染

Vue.js的核心是通过一个简洁且统一的模版来将声明式数据渲染进DOM的系统，换言之，也就是另一种形式的定义式。
