Tar 的使用事项

Mac 使用 tar 压缩，每个文件会多生成 "._" 文件副本，该文件在 mac 中解压的时候，他自己能删除这些多余的文件，而在多平台使用的时候，就会很麻烦，因此我们使用 COPYFILE_DISABLE 防止生成此类文件
